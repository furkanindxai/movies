
## Installation

1 . Go the src directory.
```bash
  cd path_to_movies
```

2 . Install dependencies
```bash
  npm i
```
3 .Specify the following environment variables:

  ```bash
  SALT_ROUNDS=10
  ACCESS_TOKEN_SECRET=pinkfloyd
  API_VERSION=v1

  ```
4 . Run the project using the following command:
```bash
  npm start
```
